## React Contacts Book App

To run the app locally.<br/>

```bash
git clone https://github.com/kacMadushan/contacts-book-in-css-modules.git
cd payment-app
yarn install
yarn run dev-server
Open http://localhost:8080/ to view it in the browser.
```
